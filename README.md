## 📖 Descripción del Proyecto

Este proyecto es un **Media Kit Digital de última generación**, diseñado específicamente para presentar la marca personal de **Dilan Berriel**. A diferencia de los típicos archivos PDF estáticos, este Media Kit es una experiencia web interactiva, moderna y bilingüe, pensada para cautivar a marcas, agencias y productoras.

El diseño sigue las tendencias más actuales del desarrollo frontend (Glassmorphism, Dark/Light Mode fluido, animaciones de alto rendimiento con Framer Motion) para garantizar una primera impresión inolvidable y profesional.

## ✨ Características Principales

*   🌗 **Dark / Light Mode Dinámico:** Transición ultra suave entre temas para adaptarse a las preferencias del usuario o de la marca.
*   🌍 **Bilingüe (Español / Inglés):** Cambio instantáneo de idioma sin recargar la página para alcanzar agencias internacionales.
*   📊 **Animaciones Interactivas:** Uso de `Framer Motion` para la aparición de estadísticas, porcentajes de audiencia y galerías al hacer scroll.
*   📸 **Galería Lightbox:** Visualización inmersiva del portafolio fotográfico a pantalla completa.
*   📄 **Sistema de Descarga de PDF Inteligente:** El proyecto está optimizado con reglas exclusivas de `@media print` para generar o descargar una versión en PDF A4 impecable, manteniendo el diseño, los colores oscuros y **los hipervínculos interactivos**.

## 🛠️ Tecnologías y Librerías

El desarrollo fue realizado utilizando un stack moderno sin necesidad de compilación pesada (Zero-Build Setup):

*   **HTML5 & CSS3** (Vanilla CSS avanzado, CSS Grid, Flexbox, Custom Properties).
*   **React 18** (Renderizado vía CDN con Babel standalone).
*   **Framer Motion** (Para todas las animaciones fluidas, scroll-triggers y transiciones).
*   **Lucide Icons** (Iconografía minimalista y escalable).

## 🚀 Uso y Visualización

Al ser un proyecto sin entorno de desarrollo forzado, se puede visualizar de forma inmediata:

1. Clona el repositorio.
2. Abre el archivo `index.html` en cualquier navegador moderno (preferiblemente a través de una extensión como *Live Server* de VS Code para cargar las fuentes correctamente).
3. Utiliza los botones flotantes de la esquina inferior derecha para alternar idiomas, temas o descargar el PDF estático de presentación.

## 📁 Estructura Principal

*   `index.html`: Núcleo de la aplicación. Contiene el HTML, el bloque de estilos integrados y la lógica de React/Babel en un solo archivo optimizado.
*   `Dilan Berriel _ Media Kit.pdf`: Versión estática en español optimizada para impresión y descarga rápida.
*   `Dilan Berriel _ Media Kit-En.pdf`: Versión estática en inglés.
*   `generador_pdf_backup.txt`: Respaldo del código original generador de PDF por renderizado en vivo (html2canvas).
